## ✨ Features

- 🎨 **Material You (MD3)** design with Light & Dark mode
- 👥 **3 Game Modes** — 2 Players, Easy AI, Hard AI (Minimax — unbeatable)
- 👤 **Pick X or O** before each match in single player
- 📊 **Scoreboard** with persistent scores across sessions
- 🎊 **Confetti, floating stars & win animations** on every victory
- 💧 **Ripple effects** and iOS-style **sliding segmented control**
- 💾 **localStorage** — scores, mode, theme all saved automatically
- 📱 Fully **responsive** — works on mobile, tablet & desktop
